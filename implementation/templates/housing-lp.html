Snapshot Summary

For this project I build a powerful yet simple model that will predict the price of a home, based on a set input of features. The original dataset had about eighty features but for simplicities sake I have reduced the number of features down to five. I took careful consideration selecting which features to implement, using many different techniques to bring the model to an acceptable performance while cutting out much of the original dataset (I can't imagine anyone would want to sit here and input 80 different features). The resulting accuracy is nearly as good as my original model which happened to place within the top 13% of all Kaggle submissions. So please try it out for yourself.
